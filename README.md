## Hi, I’m Leilany Torres Diaz 👋

I do research at the intersection of **computational neuroscience**, **precision psychiatry**, and **machine learning**, using neuroimaging and behavioral data to build quantitative models of brain–behavior relationships.

I’m especially interested in:
- Translational and computational psychiatry (rs-fMRI, task fMRI, behavior).
- Normative modeling and biomarker discovery for mood and psychotic disorders.
- Multimodal neuro data and alignment between human brains and deep neural networks.

  
I’m currently working on:
- Improving resting-state fMRI predictive modeling using deep neural networks for an upcoming conference project (updated repo coming around July).
  
On this profile you’ll find:
- A small-sample **rsFC predictive pipeline** on the UCLA CNP dataset.
- Projects in **aversive learning** and **brain–DNN alignment**.

For more details, see the pinned repositories below.
